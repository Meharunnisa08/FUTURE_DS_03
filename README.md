# 📊 Customer Retention & Churn Analysis

### 🚀 Data Science Internship – Task 3

---

## 🟡 Project Overview

This project analyzes customer churn and retention patterns using Excel.
The goal is to identify customer drop-off behavior and suggest strategies to improve retention.

---

## 🟡 Dataset Description

The dataset contains simulated customer data with the following features:

* Customer_ID
* Signup_Date
* Last_Active_Date
* Plan_Type (Free / Paid)
* Status (Active / Churned)
* Total_Purchases
* Churn (Derived column)
* Signup_Month
* Days_Active

---

## 🟡 Methodology

### 🔹 Data Preparation

* Created new columns:

  * Churn indicator (1 = churned, 0 = active)
  * Signup Month
  * Days Active (customer lifetime)

---

### 🔹 Key Metrics

* **Churn Rate:** 52.48%
* **Retention Rate:** 47.52%

---

### 🔹 Analysis Performed

* Churn by Plan Type
* Churn Trend Over Time (Monthly)

---

## 🟡 Dashboard

The dashboard includes:

* KPI metrics (Churn Rate & Retention Rate)
* Bar Chart: Churn Rate by Plan Type
* Line Chart: Churn Trend Over Time

---

## 🟡 Key Insights

* Overall churn rate is high (~52%), indicating significant customer drop-off
* Paid users show slightly higher churn than free users
* Churn fluctuates across different months
* Customers with low engagement tend to churn early

---

## 🟡 Recommendations

* Improve onboarding process for new users
* Increase early engagement
* Offer incentives to retain customers
* Monitor high-risk users
* Improve value of paid plans

---

## 🟡 Tools Used

* Microsoft Excel
* Pivot Tables
* Data Visualization (Charts & Dashboard)

---

## 🟡 Deliverables

* Excel Dashboard
* Churn Analysis Report
* Business Insights

---

## 🟡 Conclusion

This project demonstrates how customer data can be analyzed to identify churn patterns and improve retention strategies.
The insights can help businesses reduce customer loss and improve long-term growth.
