# 📊 Customer Retention & Churn Analysis

## 🟡 Project Overview

This project analyzes customer churn and retention patterns using Excel.

## 🟡 Dataset

Includes customer signup, activity, plan type, and churn status.

## 🟡 Key Metrics

* Churn Rate: 52.48%
* Retention Rate: 47.52%

## 🟡 Analysis

* Churn by Plan Type
* Churn Trend Over Time

## 🟡 Key Insights

* High churn rate (~52%)
* Paid users churn slightly more
* Churn varies across months

## 🟡 Recommendations

* Improve onboarding
* Increase engagement
* Offer retention incentives

## 🟡 Tools Used

* Microsoft Excel
* Pivot Tables
* Dashboard
